# 📚 StudyHUB

StudyHUB is a React-based study planner and task management web application designed to help students organize their study tasks, notes, and schedules efficiently.

---

## 🚀 Features

- Task creation, editing, and deletion  
- Calendar view for planning  
- Notes management system  
- Progress tracking  
- LocalStorage data saving  
- Responsive UI  

---

## 🛠️ Tech Stack

- React.js  
- JavaScript  
- HTML  
- CSS  
- Bootstrap  

---

## 📂 Project Structure

StudyHUB/
 ├── src/
 ├── public/
 ├── package.json
 ├── index.html

---

## ▶️ How to Run

1. Clone the repository  
2. Open folder  
3. Run:
   npm install  
   npm run dev  

---

## 🎯 Purpose

This project helps students manage their study tasks and improve productivity.
